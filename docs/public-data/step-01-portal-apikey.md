## Step 01 — 공공데이터포털 가입 및 API 키 발급 `(20분)`

### 이 단계에서 하는 것

정부가 운영하는 **공공데이터포털(data.go.kr)**에 가입하고,
영화진흥위원회(KOFIC) 박스오피스 API 키를 발급받습니다.
발급받은 키는 `.env.local`에 저장합니다.

---

### 1. 공공데이터포털이란

지금까지 영화 데이터는 TMDB(미국 민간 서비스)에서 가져왔습니다.
TMDB는 포스터·줄거리·평점 등 풍부한 영화 정보를 제공하지만,
**한국 박스오피스 순위**는 제공하지 않습니다.

한국 박스오피스 순위는 **영화진흥위원회(KOFIC)**가 공식 집계합니다.
KOFIC이 제공하는 API를 사용하면 매일 갱신되는 실시간 박스오피스 데이터를 앱에 표시할 수 있습니다.

**공공데이터포털(data.go.kr)**은 행정안전부가 운영하는 플랫폼으로,
KOFIC을 포함한 수백 개 공공기관의 데이터를 한 곳에서 신청·관리할 수 있습니다.

```
공공데이터포털(data.go.kr)
└── 영화진흥위원회 박스오피스 API  ← 우리가 사용할 API
└── 기상청 날씨 API
└── 교통 API
└── ... (수백 개)
```

---

### 2. 회원가입

**주소**: [https://www.data.go.kr](https://www.data.go.kr)

1. 페이지 오른쪽 상단 **회원가입** 버튼을 클릭합니다.
2. 회원 유형 선택 화면에서 **일반회원** 을 클릭합니다.
3. 이용약관 동의 → 필수 항목에 체크 → **다음** 클릭
4. 본인인증 방법 선택 (휴대폰 인증 또는 아이핀)
5. 정보 입력 (아이디, 비밀번호, 이름, 이메일 등) → **가입완료**

> 이미 계정이 있으면 로그인 후 다음 단계로 넘어갑니다.

---

### 3. KOFIC 박스오피스 API 검색

로그인 후 API를 검색합니다.

1. 상단 메뉴 **데이터찾기** 클릭
2. 검색창에 `영화진흥위원회 박스오피스` 입력 → **검색**
3. 결과 목록에서 **영화관입장권통합전산망 주간/일별 박스오피스** 항목을 클릭합니다.

> 검색 결과가 여러 개이면 **제공기관**이 `영화진흥위원회`인 항목을 선택합니다.

---

### 4. 활용신청

API 상세 페이지에서 키를 신청합니다.

1. 오른쪽 상단 **활용신청** 버튼 클릭
2. 활용목적 입력창이 나타납니다.
   - 예시: `Next.js 영화 앱 개발 학습용`
3. 계정 유형 선택:

| 유형 | 일일 한도 | 승인 방식 |
|------|-----------|-----------|
| **개발계정** | 1,000건/일 | **자동 승인 (즉시)** |
| 운영계정 | 100,000건/일 | 수동 심사 (수일 소요) |

학습 목적이므로 **개발계정**을 선택합니다. 자동 승인되므로 즉시 사용 가능합니다.

4. **신청** 버튼 클릭 → 완료

---

### 5. API 키 확인

발급받은 키는 마이페이지에서 확인합니다.

1. 오른쪽 상단 프로필 아이콘 → **마이페이지** 클릭
2. 왼쪽 메뉴 **오픈API** → **개발계정** 클릭
3. 신청한 API 목록에서 **영화진흥위원회 박스오피스** 항목의 **일반 인증키** 를 복사합니다.

> 키는 문자와 숫자로 이루어진 긴 문자열입니다.
> 예: `abc123def456...`

---

### 6. .env.local에 키 저장

프로젝트 루트의 `.env.local` 파일을 열어 아래 항목을 추가합니다.

```
# KOFIC (영화진흥위원회) 박스오피스 API Key
KOFIC_API_KEY=발급받은_키_붙여넣기
```

`.env.local` 파일 전체 예시:

```
# TMDB API Key
TMDB_API_KEY=2fa914ab1aea0e520bff8f8b50c2d92c
# NextAuth
NEXTAUTH_SECRET=...
NEXTAUTH_URL=http://localhost:3000
# PostgreSQL Database URL
DATABASE_URL=postgresql://postgres:비밀번호@localhost:5432/movie_app
# KOFIC 박스오피스 API Key
KOFIC_API_KEY=발급받은_키_붙여넣기
```

> `.env.local`은 `.gitignore`에 포함되어 있어 Git에 올라가지 않습니다.
> API 키가 공개 저장소에 노출되지 않도록 반드시 이 파일에만 저장합니다.

---

### 참고 — KOFIC 자체 포털

공공데이터포털 외에 KOFIC은 자체 개발자 포털도 운영합니다.

**KOBIS 오픈API**: [https://www.kobis.or.kr/kobisopenapi](https://www.kobis.or.kr/kobisopenapi)

두 포털 중 어느 쪽에서 발급해도 동일한 API에 접근할 수 있습니다.
이 수업에서는 공공데이터포털 경로를 사용합니다.
