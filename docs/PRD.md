# PRD — 영화 정보 앱 (Movie App)

**문서 버전**: v1.0
**작성일**: 2026-05-16
**상태**: 초안

---

## 1. Overview

### 배경

영화를 탐색하고 관심 영화를 즐겨찾기로 관리할 수 있는 영화 정보 서비스가 필요하다. 사용자는 장르 필터와 검색으로 원하는 영화를 빠르게 찾고, 상세 정보를 확인하며, 관심 영화를 저장해 언제든 다시 볼 수 있어야 한다.

### 목표

영화 탐색·검색·상세 조회·즐겨찾기 관리·로그인 기능을 갖춘 영화 정보 SPA를 구축한다.

## 3. Functional Requirements

### 3-1. 홈 페이지 (`/`)

| 기능 | 세부 요구사항 |
| --- | --- |
| Hero 슬라이더 | `isFeatured: true`인 영화 3개, 자동재생(3.5초), 페이지네이션·네비게이션 버튼, 루프 |
| 장르 필터 | 전체 / 장르별 버튼, 선택 시 목록 즉시 갱신, 영화 수 표시 |
| 영화 목록 | 그리드 레이아웃 (모바일 2열 → 데스크탑 4열), TanStack Query로 json-server에서 fetch |
| 영화 카드 | 포스터(이모지 placeholder), 제목(한/영), 개봉년도, 평점, 즐겨찾기 버튼 |
| 로딩 상태 | Shadcn UI Skeleton 카드 표시 |
| 에러 상태 | 에러 메시지 + 재시도 버튼 |

### 3-2. 검색 페이지 (`/search`)

| 기능 | 세부 요구사항 |
| --- | --- |
| 검색 폼 | React Hook Form + Zod 스키마(최소 1자 이상) |
| 검색 실행 | TanStack Query로 json-server에서 title 키워드 기반 필터 fetch |
| 검색 결과 | 영화 카드 그리드, 결과 수 표시 |
| 결과 없음 | 안내 메시지 + 홈으로 버튼 |
| 로딩/에러 | 로딩 Skeleton, 에러 메시지 |

### 3-3. 상세 페이지 (`/movies/:id`)

| 기능 | 세부 요구사항 |
| --- | --- |
| 영화 정보 | 제목(한/영), 장르, 개봉년도, 감독, 상영시간, 평점, 줄거리 |
| 즐겨찾기 | 즐겨찾기 추가/제거 버튼, Zustand favoriteStore 즉시 반영 |
| 없는 영화 처리 | 존재하지 않는 ID 접근 시 안내 메시지 + 홈으로 버튼 |
| 뒤로가기 | 이전 페이지로 이동 |

### 3-4. 즐겨찾기 페이지 (`/favorites`)

| 기능 | 세부 요구사항 |
| --- | --- |
| 목록 표시 | Zustand favoriteStore의 영화를 카드 그리드로 표시 |
| 개별 삭제 | 즐겨찾기 제거 버튼 클릭 시 해당 영화 제거 |
| 전체 삭제 | 전체 초기화 버튼 |
| 빈 상태 | 안내 메시지 + 홈으로 버튼 |

### 3-5. 로그인 페이지 (`/login`)

| 기능 | 세부 요구사항 |
| --- | --- |
| 이메일 입력 | Zod 이메일 형식 검증 |
| 비밀번호 입력 | Zod 최소 4자 이상 검증, 텍스트 마스킹 |
| 로그인 버튼 | React Hook Form handleSubmit → Mock 인증 처리 후 홈 이동 |
| Mock 인증 | 고정 계정 (test@movie.com / 1234) |
| 인증 상태 저장 | Zustand authStore + localStorage (새로고침 후에도 유지) |
| 이미 로그인 시 | `/login` 접근 시 홈(`/`)으로 리다이렉트 |

### 3-6. 공통 — Header

| 기능 | 세부 요구사항 |
| --- | --- |
| 로고 | 클릭 시 홈(`/`)으로 이동 |
| 검색 아이콘 | 클릭 시 `/search`로 이동 |
| 즐겨찾기 아이콘 | Zustand favoriteStore 수 배지 실시간 표시, 클릭 시 `/favorites` 이동 |
| 로그인 버튼 | 비로그인 상태 시 표시, 클릭 시 `/login` 이동 |
| 로그아웃 버튼 | 로그인 상태 시 표시, 클릭 시 로그아웃 후 홈 이동 |
| 고정 헤더 | 스크롤 시 상단 고정 (sticky) |

## 7. Out of Scope

아래 기능은 이 프로젝트에서 구현하지 않는다.

| 항목 | 비고 |
| --- | --- |
| 실제 결제·예매 기능 | 이 프로젝트 범위 외 |
| 영화 리뷰·댓글 | 추후 기능 확장 시 고려 |
| 관리자 페이지 | 별도 프로젝트로 분리 |
| 실제 영화 포스터 이미지 | 이모지 placeholder로 대체 |
| 실제 인증 API | Mock 인증(고정 계정)으로 대체 |
| 서버 사이드 페이지네이션 | 클라이언트 필터링으로 대체 |
| 영화 평점 입력 | 데이터 조회만 제공 |
